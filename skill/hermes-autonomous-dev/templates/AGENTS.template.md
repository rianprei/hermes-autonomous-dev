# Project

Autonomous dev follows the `hermes-autonomous-dev` skill.
Load it with: `/skill hermes-autonomous-dev`

- Use `hb` (smart) for daily work; `hb-prod` for production;
  `hb-auto` only on trusted repos.
- Never edit main/master directly — create a feature branch first.
- Checkpoints are ON — recover with `/rollback` or `git stash pop`.
- Do NOT use root/proot solutions (device constraint).
